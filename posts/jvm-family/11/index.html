阿里巴巴Arthas

Arthas 是 Alibaba 在 2018 年 9 月开源的 Java 诊断工具。支持 JDK6+， 采用命令行交互模式，可以方便的定位和诊断线上程序运行问题。Arthas 官方文档十分详细，详见：https://alibaba.github.io/arthas

参考文章：https://arthas.aliyun.com/doc/quick-start.html

1、启动math-game

wget https://arthas.aliyun.com/math-game.jar # 下载远程代码
nohup java -jar math-game.jar >msg.log 2>&1 & # 后台启动

math-game是一个简单的程序，每隔一秒生成一个随机数，再执行质因数分解，并打印出分解结果。

2、启动 arthas

wget https://arthas.aliyun.com/arthas-boot.jar # 下载包
java -jar arthas-boot.jar # 启动

选择应用进程：

$ $ java -jar arthas-boot.jar
* [1]: 12414 math-game.jar
  [2]: 12281 org.tanukisoftware.wrapper.WrapperSimpleApp

math-game进程是第 1 个，则输入 1，再输入回车/enter。Arthas 会 attach 到目标进程上，并输出日志：

3、查看dashboard

输入dashboard，按回车/enter，会展示当前进程的信息，按ctrl+c可以中断执行。

4、 通过 thread 命令来获取到math-game进程的 Main Class，

thread加上线程ID 可以查看线程堆栈；thread -b 可以查看线程死锁

thread 1会打印线程 ID 1 的栈，通常是 main 函数的线程。

jad demo.MathGame # jad 加类的全名

GC日志详解

java应用通过一些配置把程序运行过程中的gc日志全部打印出来，然后分析得到关键性指标，分析GC原因，调优JVM参数。

打印GC日志方法，在JVM参数里增加参数，%t 代表时间

-Xloggc:d:/gc-%t.log -XX:+PrintGCDetails -XX:+PrintGCDateStamps -XX:+PrintGCTimeStamps -XX:+PrintGCCause -XX:+UseGCLogFileRotation -XX:NumberOfGCLogFiles=10 -XX:GCLogFileSize=100M

Tomcat则直接加在JAVA_OPTS变量里。

详细分析：

2023-09-30T21:18:06.751+0800: 23.826: [Full GC (Ergonomics) [PSYoungGen: 53664K->22303K(79360K)] [ParOldGen: 167244K->198367K(264192K)] 220909K->220671K(343552K), [Metaspace: 3995K->3995K(1056768K)], 0.0313734 secs] [Times: user=0.23 sys=0.00, real=0.03 secs]

1、对于23.826： 这是从jvm启动开始计算到这次GC经过的时间，前面还有具体的发生时间日期。

2、Full GC(Ergonomics)指这是一次full gc，括号里是gc的原因， PSYoungGen是年轻代的GC，ParOldGen是老年代的GC，Metaspace是元空间的GC

3、[PSYoungGen: 53664K->22303K(79360K)]，这三个数字分别对应GC之前占用年轻代的大小，GC之后年轻代占用，以及整个年轻代的大小。

4、[ParOldGen: 167244K->198367K(264192K)]，这三个数字分别对应GC之前占用老年代的大小，GC之后老年代占用，以及整个老年代的大小。

5、220909K->220671K(343552K)，这三个数字分别对应GC之前占用堆内存的大小，GC之后堆内存占用，以及整个堆内存的大小。

6、[Metaspace: 3995K->3995K(1056768K)]，这三个数字分别对应GC之前占用元空间内存的大小，GC之后元空间内存占用，以及整个元空间内存的大小。

7、0.0313734是该时间点GC总耗费时间。

示例代码：

import java.util.ArrayList;

public class HeapTest
{
    // 100KB
    byte[] a = new byte[1024 * 100];
    
    public static void main(String[] args)
        throws InterruptedException
    {
        ArrayList<HeapTest> heapTests = new ArrayList<>();
        while (true)
        {
            heapTests.add(new HeapTest());
            Thread.sleep(10);
        }
    }
}

JVM参数汇总查看命令

java -XX:+PrintFlagsFinal 表示打印出所有参数选项在运行程序时生效的值
